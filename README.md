# LANDING PAGE

Landing page desenvolvida para a empresa S.I Alinhamentos, especializada em alinhamento de caminhões e serviços automotivos pesados.

O projeto foi desenvolvido utilizando HTML, CSS e JavaScript puro, com foco em design responsivo, apresentação dos serviços e contato rápido com clientes.

Funcionalidades:
- Apresentação da empresa
- Exibição dos serviços oferecidos
- Layout responsivo para mobile e desktop
- Seção de contato para clientes
- Carregamento rápido com tecnologias web puras

Tecnologias Utilizadas:
- HTML5 — Estrutura da página
- CSS3 — Estilização e layout responsivo
- JavaScript (ES6) — Interatividade da interface

Deploy:
  Este projeto está hospedado utilizando GitHub Pages.
